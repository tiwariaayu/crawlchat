# Chrome Extension

The CrawlChat Chrome Extension lets you generate text content from your documentation directly in any text field across the web. Write support emails, product descriptions, or social posts using your own knowledge baseâ€”no copy-pasting needed.

### Setup

#### 1. Install the Extension

Install the CrawlChat Chrome Extension from the [Chrome Web Store](https://chromewebstore.google.com/detail/crawlchat/icimflpdiioobolkjdbldmmomflainie).

#### 2. Get Your API Key

1. Go to [API Keys](https://crawlchat.app/api-key) on your dashboard
2. Create a new API key
3. Copy the key

#### 3. Configure the Extension

1. Open the extension
2. Paste your API key
3. Save it
4. Select your collection (workspace)
5. (Optional) Add custom prompt instructions

### How to Use

#### Method 1: Copy text

1. Click on the left bottom CralwChat sticky button
2. Explain the text to be written about
3. Iterate for fine tuning
4. Copy the text

#### Method 2: Use it

1. Click on any text field
2. Enter `@` following by any instruction for the text to be created. Ex: `@ write about pricing`
3. Click on the sticky button
4. Generate it
5. Click `Use it` button to insert it into the text field

#### Method 3: Instant

1. Click on any text field
2. Enter `@` following by any instruction for the text to be created. Ex: `@ write about pricing`
3. Press **Cmd + Shift + P** (Mac) or **Ctrl + Shift + P** (Windows)
4. Takes few seconds and the text will be replaced by the created text

#### Method 4: Context menu

1. Click on any text field
2. Give the instruction to compose. Ex: `write about installation under 50 words`
3. Right click on the text field
4. Go to `CrawlChat`
5. Select `Compose` to **Use it** as mentioned above
6. Select `Fill` for **Instant** fill as mentioned above

### Requirements

- Active CrawlChat account
- Valid API key
- At least one collection with documentation
