# Chrome Extension

The CrawlChat Chrome Extension lets you generate text content from your documentation directly in any text field across the web. Write support emails, product descriptions, or social posts using your own knowledge base—no copy-pasting needed.

## Key Features

### Text Generation Anywhere
- Works in any text box on any website
- Generates content from your documentation
- Creates reliable, context-aware responses
- Not generic AI—uses your actual docs

### Iterative Refinement
- Start with a prompt: "Write about pricing plans"
- Refine iteratively: "Add purchase links"
- Keep improving: "Make it shorter"
- Maintains context throughout the conversation

### Keyboard Shortcuts
- **Cmd + Shift + P** (Mac) / **Ctrl + Shift + P** (Windows)
- Instantly triggers content generation
- Auto-inserts content when ready
- No need to open popup manually

### Custom Prompts
Configure default instructions for your content:
- "Keep content short"
- "Use bullets as much as possible"
- "Don't use markdown"

### Multi-Collection Support
- Access all your CrawlChat collections
- Switch between different knowledge bases
- Perfect for managing multiple products or teams

## Setup

### 1. Install the Extension
Install the CrawlChat Chrome Extension from the Chrome Web Store.

### 2. Get Your API Key
1. Go to [API Keys](https://crawlchat.app/api-key) on your dashboard
2. Create a new API key
3. Copy the key

### 3. Configure the Extension
1. Open the extension
2. Paste your API key
3. Save it
4. Select your collection (workspace)
5. (Optional) Add custom prompt instructions

## How to Use

### Method 1: Button Click
1. Click in any text field
2. A CrawlChat button appears
3. Type your request in the text field
4. Click the CrawlChat button
5. Review and refine the generated content
6. Click "Use it" to insert

### Method 2: Keyboard Shortcut
1. Type your request in any text field
2. Press **Cmd + Shift + P** (Mac) or **Ctrl + Shift + P** (Windows)
3. Content auto-generates and inserts immediately

## Behind the Scenes

The extension uses the CrawlChat Compose API to:
- Pull information from your knowledge base
- Generate contextual responses
- Maintain conversation threads
- Apply your custom formatting preferences

All queries appear in your dashboard under the API channel for tracking and analytics.

## Requirements

- Active CrawlChat account
- Valid API key
- At least one collection with documentation
